## Overview

Binance AI Bot is a modern, modular and scalable cryptocurrency trading platform built with Python.

The project combines:

- AI-powered market analysis
- Automated trading strategies
- Advanced risk management
- Technical indicators
- Backtesting engine
- Telegram notifications
- Docker deployment
- REST API
- Modular architecture

Designed for developers, traders and researchers who want to build professional automated trading systems.
